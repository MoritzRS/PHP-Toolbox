# PHP Toolbox

Basic setup to get PHP projects up and running without any framework.
Includes tests to ensure stability.

## Target

- Shared Hosting (no composer)
- Apache
- PHP7
- SQLite3

## Features

- [x] Autoloader
- [x] Global Config
- [x] Setup Scripting
- [x] Testing Utils
- [x] Database Utils
- [x] Router
- [x] JWT Util
- [x] Templates (Validator, Normalizer)
- [x] File Upload Handling
